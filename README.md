## üìñ How It Works

### Encryption Process
1. **Character Shifting**: Each character's Unicode value is increased by a salt value (13)
2. **Base64 Encoding**: The shifted string is converted to Base64
3. **Prefix Addition**: `SC_` prefix is added to identify valid encrypted strings

### Decryption Process
1. **Prefix Validation**: Checks for `SC_` prefix
2. **Base64 Decoding**: Converts from Base64 back to shifted string
3. **Character Restoration**: Each character's Unicode value is decreased by salt (13)

### Technical Implementation
```javascript
// Original algorithm (preserved from your code)
function customEncrypt(text) {
    let shifted = "";
    for (let i = 0; i < text.length; i++) {
        shifted += String.fromCharCode(text.charCodeAt(i) + 13);
    }
    return "SC_" + btoa(unescape(encodeURIComponent(shifted)));
}
```
